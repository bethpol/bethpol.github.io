Featured Project: Minion Chat

After re-watching Despicable Me since it was put on Netflix, I figured a cute minion-themed chatroom would be the perfect way to learn about websockets. I learned about how to use websockets to create websites that communicate to the server, with the ability for the server to then communicate back to the other clients. To implement websockets for my own purposes, I used an express server running a websocket server with the websocket.io library.
